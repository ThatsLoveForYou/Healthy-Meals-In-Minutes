# 🥗 Healthy Meals in Minutes
### A TLFY Wellness System Tool

**Healthy Meals in Minutes** is an interactive web application designed for busy professionals, parents, and health enthusiasts who need nutritious meal ideas without the stress of long prep times.

## 🚀 Features
* **Instant Inspiration**: One-click generation of healthy, balanced meal ideas.
* **Time-Focused**: Every suggestion includes a prep time to fit your schedule.
* **Wellness Tips**: Each meal comes with a quick "why" behind the nutrition (e.g., Omega-3s, High Protein).

## 💡 Why it exists
This tool is part of the **That’s Love For You (TLFY)** mission to simplify wellness and provide actionable systems for a healthier lifestyle.

## 🛠️ Tech Stack
* **HTML5/Tailwind CSS** for a clean, responsive mobile design.
* **JavaScript** for the logic engine.
* **GitHub Pages** for seamless deployment.
